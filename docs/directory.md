### ディレクトリ構成

- app

  - (pages)
    ページのルーティングとなる。

- api

  - api
    API について記述する。

- components
  共通のコンポネント

- features
  固有のコンポネント。固有に定義が必要なものもここに含まれる。

- hooks
  共通のフック。(エラーハンドリングなど)

- states
  共通の Recoil State

- docs
  ドキュメント関連

- constants
  共通の定数

- lib
  関数(主にバックエンド側)

- public
  画像など

- utils
  共通の関数。(主にクライアント側)

- types
  共通の型定義

- styles
  デザイン周りはここ。
